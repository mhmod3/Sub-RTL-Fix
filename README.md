
# أداة إصلاح اتجاه النص في ملفات الترجمة (RTL Fixer)

أداة ويب بسيطة وقوية لمعالجة ملفات الترجمة (SRT, VTT, ASS) وإصلاح مشكلة اختلاط النصوص العربية (التي تكتب من اليمين إلى اليسار) مع النصوص اللاتينية أو الأرقام، مما يؤدي إلى تشوه العرض في العديد من المشغلات والتطبيقات.

---

## المشكلة

عند مزج النصوص العربية مع علامات ترقيم أو أرقام أو كلمات إنجليزية داخل ملف ترجمة، قد يظهر النص معكوساً أو مشوشاً بسبب عدم دعم التطبيق المستخدم للاتجاه الصحيح (RTL). مثال:

**النص الأصلي (قبل الإصلاح):**
```
1
00:00:05,000 --> 00:00:10,000
مرحبا بك في البرنامج التعليمي رقم 3
```
**كيف يظهر في بعض المشغلات:**  
`3 رقم التعليمي البرنامج في بك مرحبا` (معكوس)

---

## الحل

تقوم الأداة بما يلي:
1. **كشف ترميز الملف** تلقائياً (UTF-8, Windows-1256, ISO-8859-6, UTF-16, ...) وفك تشفيره بشكل صحيح.
2. **تحديد صيغة الملف** (SRT, VTT, ASS) تلقائياً.
3. **تطبيق أحكام اتجاه Unicode** (`U+202B` RIGHT-TO-LEFT EMBEDDING و `U+202C` POP DIRECTIONAL FORMATTING) على **النصوص فقط** (أي أسطر الترجمة)، مع **الحفاظ على التوقيتات والأرقام والوسوم** (مثل HTML أو `{..}` في ASS) دون تعديلها.
4. **إزالة أي أحكام اتجاه سابقة** لتجنب التعارض.
5. **إخراج الملف** بصيغة UTF-8 جاهز للاستخدام.

---

## مثال توضيحي

### ملف SRT قبل المعالجة
```
1
00:00:02,398 --> 00:00:09,590
<font color="#fffc00">مرحبا بك في الدرس الأول</font>

2
00:00:09,590 --> 00:00:16,783
<font color="#fffc00">هذا مثال على النص المختلط مع رقم 123</font>
```

### بعد المعالجة (ما تراه في الملف الناتج)
```
1
00:00:02,398 --> 00:00:09,590
<font color="#fffc00">‫مرحبا بك في الدرس الأول‬</font>

2
00:00:09,590 --> 00:00:16,783
<font color="#fffc00">‫هذا مثال على النص المختلط مع رقم 123‬</font>
```
> ملاحظة: الأحرف الخاصة `‫` و `‬` هي أحكام اتجاه غير مرئية تضمن عرض النص بالاتجاه الصحيح.

---

## الميزات الرئيسية

- **دعم صيغ متعددة**: SRT, VTT, ASS.
- **كشف ترميز تلقائي**: يتعامل مع معظم الترميزات الشائعة.
- **الحفاظ على الوسوم**: لا تتأثر علامات HTML أو أكواد ASS مثل `{..}`.
- **الحفاظ على التوقيتات**: تبقى أرقام التوقيت والفواصل الزمنية كما هي.
- **واجهة سهلة**: سحب وإفلات، زر معالجة، تحميل فوري.
- **خط Adobe Arabic** للعرض الواضح (اختياري).

---

## كيفية الاستخدام

1. افتح الصفحة في المتصفح.
2. اسحب ملف الترجمة (SRT, VTT, ASS) إلى المنطقة المحددة أو اختره عبر الزر.
3. سيتم عرض معاينة للنص بعد فك الترميز.
4. اضغط على زر **"معالجة وإصلاح"**.
5. ستظهر النتيجة في المربع السفلي.
6. يمكنك **تحميل الملف المُصلح** أو **نسخ النص** مباشرة.

---

## التقنيات المستخدمة

- **الواجهة**: HTML + CSS (تصميم داكن، متجاوب).
- **المنطق**: JavaScript (ES6) مع استخدام `TextDecoder` لفك الترميز.
- **معالجة النصوص**: تعزيز دوال `String.prototype` لإضافة `toRTL()` و `clearFormatting()`.
- **التوافق**: يعمل في جميع المتصفحات الحديثة.

---

## ملاحظات للمطورين

- يمكن دمج الأداة كـ `iframe` أو تضمينها في مشاريع أخرى.
- كود المعالجة الأساسي موجود في دالة `fixSubtitleFile()`.
- يمكن تعديل السلوك عبر تغيير دوال المعالجة الخاصة بكل صيغة.
